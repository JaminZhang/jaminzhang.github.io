Python 基础 01-常量和变量


# 常量

常量：值不能改变的量。


## 数字常量：1 100 3.5

* 整数类型常量
* 浮点数类型常量

## 字符串常量： 'a' "123" '''可以跨越多行'''

* 单引号
* 双引号
* 三引号

## 布尔值常量：True False

首字母大写

## 高级类型常量: [] {}

* 列表：[]
* 字典：{}

# 变量

变量：值能够改变的量。

通过等号改变它的值，也就是赋值。

```python

a = 1
a = 20		# 改变了值
a = "hi"	# 改变了数据类型

```    

## 变量命名

* 只能包含数字、字母和下划线
* 数字不能开头	12name 是非法的变量名
* 区分大小写	Aname 和 anmae 是不同的变量
